# SYNTACXHUB Weather App

A responsive Weather Forecast Application built using React.js, Vite, Tailwind CSS, Axios, and OpenWeather API.

## Features

* Search weather by city name
* Real-time weather information
* 5-Day weather forecast
* Dynamic weather icons
* Dynamic background based on weather conditions
* Loading spinner
* Error handling
* Responsive design
* Enter key search support

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* Axios
* OpenWeather API

## Installation

Clone the repository:

git clone https://github.com/ashutoshmohanty1/syntacxhub-weather-app.git

Install dependencies:

npm install

Create a .env file:

VITE_WEATHER_API_KEY=YOUR_API_KEY

Run the application:

npm run dev

Build for production:

npm run build
