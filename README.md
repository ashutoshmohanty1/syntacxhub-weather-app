# SYNTACXHUB Weather App

## Overview

A modern Weather Forecast Application built using React.js, Tailwind CSS, Axios, and OpenWeather API. The application provides real-time weather information and a 5-day weather forecast for any city.

## Features

* Search weather by city name
* Current weather conditions
* 5-day weather forecast
* Dynamic weather icons
* Dynamic background based on weather conditions
* Loading spinner
* Error handling
* Responsive design
* Enter key search support

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* Axios
* OpenWeather API

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Create a .env file

```env
VITE_WEATHER_API_KEY=YOUR_API_KEY
```

4. Run the project

```bash
npm run dev
```

## Build Project

```bash
npm run build
```
